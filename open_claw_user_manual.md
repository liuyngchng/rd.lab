# 1. 说明

本文档面向非技术人员，其无法通过 `OpenClaw` 官网操作说明自主完成 `OpenClaw` （俗称龙虾）的安装。 

# 2. 用户操作

 本文档有相应的配套操作演示视频，用户以观看视频为主，文档为辅，完成 `OpenClaw`的配置和使用。
## 2.1 充值大模型（必需项）

本文档以`OpenClaw` 官方支持的中国大模型`MimiMax` 为例进行说明。在浏览器地址栏中输入

```
https://platform.minimaxi.com/login
```



`MimiMax` 是 openclaw 默认的大模型之一，这个需要充值，不然  `OpenClaw`  无法使用，充个10块8块的就行。

用户的大模型 API（接口）账户里头有钱了，就具备操作下一步的条件了。

接下来创建个接口密钥，即 API key    

```
sk-api-5qo1kB8QXCHHvZLa3NGhMfCPeDsf3EzXHYt6...
```

好了，这个密钥，自己拷贝到一个记事本上保存下来，等待后续 配置   `OpenClaw` 的时候，会用到。

## 2.2 配置`IM`（即时通信）（可选项）

这里以 `OpenClaw` 官网支持的`Feishu(飞书)`为例进行说明。

配置 `Feishu`，是为了通过手机` APP` 远程控制 龙虾`OpenClaw`在自己的电脑上工作。如果没有远程控制的需求，该章节可以忽略。不配置` IM`， 不会影响 `OpenClaw`的主要功能。

**（1）登录 `Feishu`开发者平台** 

```
https://open.feishu.cn
```

用户登录后可能需要创建个组织什么的，随便创建一个就行。

**（2）配置开发者平台**

**1）创建自建应用。**首先创建一个应用，应用名称随便起一个自己喜欢的。

**2）选择图标。**应用图标选个自己喜欢的，也可以用默认的。

**3）添加机器人功能。**给应用添加机器人功能，点击添加应用能力。添加机器人功能后，在应用的左侧导航栏有机器人这个条目了。

**4）配置权限。**接下来，开始给机器人配置飞书的权限，主要包含`im:`和`contact:`两部分，即即时消息和通讯录使用。

**（3） 发布应用**

配置好自建应用之后，需要进行发布。带*号的是必须填写的，不填写发布不了。

**（4）** **获取 `APP_ID`, `APP_KEY`**。 来自飞书开发者平台。 龙虾(`OpenClaw`) 通过 `APP_ID` 和  `APP_KEY` 就能跟 飞书建立联系了。



#  3. `OpenClaw` 部署

完成章节 2. 的工作后， 就交由技术人员进行部署。具体采用的方案为 USB外接启动的Ubuntu系统，通过在现有电脑的BIOS启动选项中，设置从USB启动，读取 USB设备中的外接系统，实现与当前工作环境的安全物理隔离。

安装完成后的系统信息如下：

```sh
操作系统： Ubuntu 24.04 LTS （乌班图 24.04 长期支持版）
操作系统用户名：openclaw
操作系统登录密码：openclaw
```



# 4. 用户配置

## 4.1 登录 Ubuntu 系统

当用户拿到已经安装好 `OpenClaw` 的 `USB` 设备后，接入自己的电脑，在电脑启动时，选择从`USB`启动，这个步骤很重要，具体需要按照自己计算机的主板厂商说明进行， 有的是按住`F12`, 有的是按住`DEL`键。需要进入 `BIOS/UFEI` 启动项选择，如果`BIOS` 设置了密码还需要输入密码，选择自己外接的 `USB` 设备符号。

进入 `Ubuntu` 系统 boot 界面后，很快会进入系统（有个10s的倒计时，用户等待即可）， 系统已经设置为免密登录。

## 4.2 为 `OpenClaw`配置大模型 

进入 `Ubuntu`系统后，需要为`OpenClaw`配置 大模型`API`才能正常工作，下面将通过视频演示如何操作。

**（1） 获取口密钥**

网页浏览器地址栏中输入这个网址

```
https://platform.minimaxi.com/login
```

手机验证码登录，保证自己的账户余额大于0，不然这软件没法用
开始创建接口密钥

```
sk-api-VLyLLWMlN2f....
```

**（2）启动配置界面**

打开 `Terminal`（终端），类似 `Windows` 中的 `CMD`

输入以下文本

```sh
openclaw onboard
```

**（3）开始配置**

注意，在操作过程中，任何时候出现操作错误，直接关闭 `Terminal` 后， 重新打开 `Terminal` 输入

```
openclaw onboard
```

重新开始。

1）使用键盘的左右键，选择 “Yes”，输入回车，进入下一步；

2）选择 “`QuickStart`”, 输入回车键，进入下一步；

3）选择 “`MiniMax`” ，当然你可以选择任何你喜欢的大语言模型，我这里以 `MimiMax` 做个示范。输入回车键，进入下一步；

4）在中国大陆地区，选择 "`MiniMax CN- API Key`"，输入回车键，进入下一步；

 5）这里， 粘贴你的接口密钥，这个非常重要，一定要输入正确，输入回车键，进入下一步；

6）通过键盘上的空格键（Space），选择 "Skip for now" ， 跳过飞数控制配置选项，进入下一步；

对于新手来说，飞书控制的配置略复杂，如果你明白其中的道理，而且已经完成了步骤2.2 中的操作，那么这里可以进行相关的配置。不使用飞书远程控制，不影响 `OpenClaw `主要功能的体验。输入回车键，进入下一步；

7）等待即可，看到了选择在哪里作为使用入口的步骤，如果你喜欢模仿一些极客（`Geek`），在一块黑漆漆的界面上操作，那么这里你可以选择 "`Open in TUI`"，意思是 Terminal User Interface， 终端用户界面。

我这里的演示使用常规的浏览器作为`OpenClaw`的使用入口， 所以这里选择  "`Open the Web UI`" 这个选项。输入回车键，进入下一步；

8）进入浏览器使用界面。好了，现在你可以像使用一个正常的聊天软件与 `OpenClaw` 进行通话了。



在这个过程中，必须保证所使用计算机的网络连接通常，可以自由访问国内的公共互联网，不要使用任何代理。


# 5. 常规使用

对于已经配置好大语言模型`API` 密钥的用户来说，开机即可使用。如果没有配置大语言模型API，请查阅章节4. 中的说明，一步一步操作。后续操作只需要按照如下步骤即可。

（1）电脑开机，进入`Ubuntu`系统；

（2）打开 `Terminal`；

（3）输入

```sh
openclaw dashboard
```

（4） 在浏览器中开始对话；

（5）使用完后，关闭浏览器，关闭机器。
